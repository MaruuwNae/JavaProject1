# Project 1 - Spring MVC (Thymeleaf)

This project is a simple Spring MVC application that renders web pages using
Thymeleaf templates. It demonstrates how controllers return views and how
parameters from the URL are displayed on a web page.

# How to Run

1. Open the project in IntelliJ IDEA.
2. Run the main class: FirstProjectSpringApplication
3.  Open a browser and go to: http://localhost:8080/
